リウマトレックスカプセル2mg インタビューフォーム（第4版, 2024年2月）＋ MTX使用と診療の手引き2023年版（日本リウマチ学会）に基づく

MECHANISM まず理解しておく作用機序と検査値への影響の全体像
①葉酸拮抗作用（DHFR阻害）
ジヒドロ葉酸還元酵素（DHFR）を阻害 → テトラヒドロ葉酸（活性型葉酸）産生↓ → DNA合成・細胞分裂を抑制。
⚠ 造血細胞・消化管粘膜など増殖速度の高い細胞に影響が出やすい → 血液毒性・消化管障害の背景機序。MCVの上昇（大球性変化）は葉酸拮抗の早期指標となりうる。
②アデノシン放出促進（抗炎症作用）
AICAR transformylaseの阻害 → 細胞外アデノシン増加 → 好中球遊走・炎症性サイトカイン産生・血管新生を抑制 → 関節炎の活動性が低下。
→ CRP・ESR・MMP-3の低下はこの機序による治療効果の反映として解釈する。
③肝臓・腎臓との関係
MTXは主に腎臓から未変化体で排泄（尿中69.5%）される。腎機能が低下するとMTXが体内に蓄積し、骨髄抑制・肝障害など全副作用リスクが急激に増大。アルブミン低下 → 遊離型MTX増加 → 同様の毒性増強。

→ 検査値の変動には「治療効果として動く炎症指標の改善」と「薬の毒性として動く血液・肝・腎指標の悪化」が混在する。週1回投与であることを念頭に、変動の文脈を判断することが臨床検査技師の報告の要点となる。

EFFICACY 治療効果を評価する検査とその解釈
CRP・ESR（赤血球沈降速度） 低下＝改善 主要炎症指標
なぜ両方測るか
CRPは急性相タンパクとして炎症に迅速に反応（数時間〜1日）。ESRはフィブリノゲン増加を反映し変動が緩やか（数日〜週単位）。DAS28スコア算出にはCRP版とESR版の2種類が存在し、どちらが使われているか把握しておくと依頼の意図が分かりやすい。

モニタリング頻度（手引き2023年版）：MTX開始後あるいは増量後3カ月以内は2〜4週ごと、以降は4〜12週ごと。

⚠ MTX投与中に炎症改善なくCRPが上昇している場合、疾患活動性の再燃のほか感染症（日和見感染を含む）の可能性も高い。CRP上昇を見たら単純に「疾患悪化」とは即断せず、臨床側との情報共有が有用。

低下・陰性化 → 治療奏効 横ばい → 増量・他薬追加を検討 再上昇 → 再燃 or 感染症を鑑別
MMP-3（マトリックスメタロプロテアーゼ-3） 低下＝改善 12週ごと算定
何を反映するか
滑膜細胞が産生する軟骨分解酵素。CRPが陰性化していてもMMP-3が高値の場合は滑膜炎が持続している可能性がある。DAS28-CRP・腫脹関節数との相関が報告されており（安田・小原, 昭和大学藤が丘病院, 臨床リウマチ Vol.24 No.2）、関節破壊リスクの補助的指標として位置付けられる。

保険上の注意：MMP-3の算定は12週（約3カ月）ごとまでしか認められていない。頻回な依頼が来た場合は査定対象になる可能性があるため、依頼側への情報提供として把握しておく価値がある。

低下継続 → 滑膜炎の鎮静化 CRP正常化後も高値持続 → 画像評価（X線・エコー）と合わせて判断
RF（リウマトイド因子）・抗CCP抗体 緩徐に低下 24〜52週ごと

炎症指標（CRP・ESR）と異なり、免疫学的指標は治療効果の反映が遅い。投与開始から数カ月〜1年以上かけて緩徐に低下することがある。短期間での評価は意義が低い。

抗CCP抗体は予後不良因子として投与前スクリーニングで評価されるが、治療効果モニタリングとしての頻回測定は通常行わない。

関節単純X線（手・足部）/ HAQ-DI 年1回 / 6〜12カ月ごと

骨びらん・関節裂隙狭小化の進行抑制をmSharp法等で評価。有効な治療下では骨破壊進行が抑制される（構造的寛解）。

HAQ-DI（Health Assessment Questionnaire Disability Index）は患者自己申告型の身体機能評価。検査室からの直接的関与は少ないが、検査値改善と身体機能改善のタイムラグを理解しておくと全体像が把握しやすい。

SAFETY 副作用を評価する検査・発現傾向・判断基準

★ 発現傾向はリウマトレックスIF（第4版）・MTX診療手引き2023年版・大阪大学呼吸器免疫内科資料より。重大な副作用は特に注意。

末梢血液検査（CBC）/ WBC・好中球・RBC・PLT・MCV 重大 死亡報告34.6%が血液障害
骨髄抑制の機序
DHFR阻害 → 造血幹細胞のDNA合成障害 → 汎血球減少・無顆粒球症。葉酸の消耗とMTX蓄積が原因。腎機能低下・低アルブミン血症があると急激に進行しうる。

国内死亡報告（2012年まで、MTX適正使用情報Vol.18 / 大阪大学）：MTX使用RA患者465例の死亡のうち血液障害161例（34.6%）が最多。汎血球減少97例、骨髄抑制39例、白血球減少18例など。

MCVに注目：葉酸拮抗により赤血球の大球性変化（MCV上昇）が骨髄抑制よりも早期に出現することがある。葉酸製剤（ロイコボリン等）を並行して処方されていることも多い。

WBC < 4,000/mm³ → 慎重経過観察 WBC < 3,000/mm³ または PLT < 50,000/mm³ → 禁忌水準・緊急報告
REPORTING POINT
前駆症状として「発熱・咽頭痛・インフルエンザ様症状」が先行することがある旨、添付文書に記載あり。WBC急低下時は感染症の合併も念頭に置いた情報提供が有用。

モニタリング頻度：開始・増量後6カ月以内は毎月、安定後は3カ月ごと（MTX手引き2023）

肝機能（AST・ALT・ALP・γGTP・アルブミン） 重大 頻度高い副作用
肝障害の機序
MTXが肝細胞内でポリグルタミン酸化されて蓄積 → 直接的な肝細胞毒性 → AST・ALT上昇。長期投与では肝線維化・肝硬変のリスクがある。飲酒との相乗効果が顕著。

臨床試験データ（リウマトレックスIF）：国内臨床試験（204例）で臨床検査値異常発現率は23.8%（15/63例）。最多は肝機能関連（ALT・AST・ALP上昇）。

アルブミンの重要性：アルブミン低下（特に<3.0 g/dL）はMTX遊離型増加につながり、毒性リスクの直接的指標になる。MTX使用手引き2023では「低アルブミン血症は慎重投与基準」として明示されている。

HBV・HCVキャリアの注意：MTX投与中および投与終了後も肝炎ウイルスマーカー（HBs抗原・HBc抗体・HBV-DNA等）のモニタリングが必要。HBV再活性化による重篤な肝炎・死亡例が報告されている（添付文書）。

AST/ALT が基準値上限の2倍超 → 減量・中止検討の目安 急激な肝機能悪化 + アルブミン低下 → 毒性増強リスクとして緊急対応

γGTPは飲酒（アルコール）でも上昇するため、問診情報との照合が重要。

腎機能（BUN・Cr・eGFR）/ 尿一般検査 重大 毒性増強の引き金
腎機能低下がなぜ危険か
MTXは腎排泄（尿中69.5%）のため、eGFR低下 → MTX蓄積 → 骨髄抑制・肝障害などあらゆる副作用が急速に増悪する。「腎機能が落ちると全ての副作用が激化する」という連鎖を理解しておくことが重要。

投与禁忌水準（MTX手引き2023）：GFR < 30 mL/分/1.73m² は投与禁忌の目安。GFR < 60 mL/分/1.73m² でも慎重投与とされ低用量から開始。

尿一般検査（蛋白・潜血）も腎障害の早期検出に使用。

eGFR 30〜60 → 慎重投与・頻回モニタリング eGFR < 30 または急激なCr上昇 → 禁忌水準・緊急報告

脱水・NSAIDs併用・造影剤使用などで腎機能が一時的に悪化した際も、MTX蓄積リスクとして臨床側に伝えることが有用。

KL-6・SP-D / 胸部X線・胸部CT 重大 死亡報告28.2%が肺障害
MTX肺炎の特徴
過敏反応が主な発症機序と考えられているが詳細は不明。用量非依存性（少量でも発症しうる）。国内調査では発症の75%が服用開始半年以内に集中しているが、数年〜十数年後の発症例も存在する（日本呼吸器学会 薬剤性肺障害手引き2013）。

国内死亡報告（大阪大学）：死亡465例中肺障害131例（28.2%）、うち間質性肺炎120例。血液障害に次いで多い。

リスク因子：60歳以上・胸膜病変・低アルブミン血症・糖尿病・既存の肺病変（Ann Intern Med 1997;127:356-64）。

KL-6・SP-Dの位置付け：MTX手引き2023では「参考程度にとどめる」と記載されており、スクリーニングの主体は症状（発熱・乾性咳嗽・呼吸困難）と胸部画像にある。KL-6高値は情報提供として有用だが単独での中止判断根拠にはしない。

KL-6・SP-D上昇 → 参考情報として報告、画像評価を促す 38℃以上の発熱 + KL-6急上昇 → MTX肺炎を積極的に疑う
REPORTING POINT
MTX投与中の発熱・呼吸器症状時にCRPが上昇していた場合、感染性肺炎・ニューモシスティス肺炎・MTX肺炎の三者鑑別が必要になる。β-Dグルカン（ニューモシスティス肺炎）などの同時依頼も念頭に。

胸部X線は年1回の定期モニタリングとして依頼される。
感染症マーカー（β-Dグルカン・CMV抗原・結核関連） 重大 死亡報告17.8%が感染症
免疫抑制と日和見感染
MTXの免疫抑制作用により、通常では発症しない病原体による日和見感染症リスクが上昇。特にニューモシスティス肺炎・サイトメガロウイルス感染症・帯状疱疹・非結核性抗酸菌症（MAC）が重要。

国内死亡報告（大阪大学）：死亡465例中感染症83例（17.8%）。ニューモシスティス肺炎25例、肺炎19例、敗血症11例など。

投与前スクリーニング：結核（Tスポット・ツベルクリン反応・胸部X線）、HBs抗原・HBc抗体・HBs抗体、HCV抗体は投与前に必須評価。

β-Dグルカン上昇 + 発熱・呼吸器症状 → ニューモシスティス肺炎として対応
可溶性IL-2受容体（sIL-2R）/ LDH・CBC 重大 平均7年後に発症
リンパ増殖性疾患（LPD）のリスク
MTX投与中のEBV関連リンパ増殖性疾患（悪性リンパ腫等）。MTX中止で自然退縮する例があることが特徴。60〜70歳代の女性に多く、MTX開始から発症までは平均7年（範囲は広い）とされる（ゼルヤンツ・LPD資料）。

初発症状と検査：リンパ節腫大が最多。発熱・寝汗・体重減少（B症状）が約30%に出現。検査上では関節症状の増悪を伴わないCRP上昇・sIL-2R高値が手がかりとなる。

sIL-2Rの解釈注意点：炎症でも上昇するため、関節炎の活動性が落ち着いているにもかかわらずsIL-2Rが高値という乖離パターンがLPDを疑う重要な手がかりになる。

sIL-2R高値 + 関節炎が落ち着いている → LPDの可能性を情報提供 LDH上昇 + リンパ球増多 + sIL-2R高値 → 悪性リンパ腫として精査

単独では解釈困難なため、他の指標との組合せと臨床情報を合わせて報告することが重要。
